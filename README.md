# 📈 Data Analyst Job Market Analysis (2026)

## Project Overview
This project analyzes the current job market trends for Data Analysts. Using a dataset of over 2,000 job postings, I identified the most in-demand skills and top locations for hiring.

## 🛠️ Tools Used
* **Python** (Pandas for cleaning, Matplotlib/Seaborn for charts)
* **Jupyter Notebooks** (VS Code)
* **Dataset:** Google Search Job Postings

## 🔍 Key Insights
* **Top Skill:** SQL is required in over 50% of entry-level roles.
* **Work Trend:** Remote roles ("Anywhere") are significantly more frequent than city-specific roles.
* **Top Hubs:** The United States remains the leading market for these positions.
